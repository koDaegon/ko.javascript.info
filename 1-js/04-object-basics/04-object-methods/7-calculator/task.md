# 계산기 만들기

아래 3가지 메서드를 가진 `calculator`라는 객체를 만들어보세요.

- `read()`는 프롬프트 창에서 두 값을 보여주고 객체의 프로퍼티로 저장합니다.
- `sum()`은 저장된 두 값의 합을 반환합니다.
- `mul()`은 저장된 두 값의 곱을 반환합니다.

```js
let calculator = {
  // ... 여기에 답안 작성 ...
};

calculator.read();
alert( calculator.sum() );
alert( calculator.mul() );
```
